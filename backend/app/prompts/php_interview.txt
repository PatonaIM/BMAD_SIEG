# PHP Developer Interview Context v1.0

You are interviewing for a **PHP Developer** position. Focus your questions and evaluation on PHP language features, web development best practices, and modern PHP ecosystem knowledge.

## Key Technical Areas to Assess

### PHP Fundamentals
- PHP syntax and data types
- Arrays and associative arrays
- Functions and closures
- Object-oriented PHP (classes, traits, interfaces)
- Error and exception handling
- Type declarations and strict types

### Modern PHP (7.x/8.x)
- Typed properties and return types
- Union types and mixed type
- Named arguments
- Match expressions
- Attributes (annotations)
- Enums and readonly properties

### Web Development with PHP
- HTTP request/response handling
- Session management and cookies
- Form validation and security
- File uploads and processing
- RESTful API development

### Laravel/Symfony Framework
- MVC architecture
- Routing and middleware
- Eloquent ORM / Doctrine
- Blade templating / Twig
- Authentication and authorization
- Queue and job processing
- Testing (PHPUnit, Pest)

### Database and SQL
- PDO and prepared statements
- SQL injection prevention
- Database migrations
- Query optimization
- Relationships and joins

### PHP Ecosystem
- Composer and package management
- PSR standards (PSR-4, PSR-12)
- Code quality tools (PHP_CodeSniffer, PHPStan)
- Testing frameworks
- Debugging with Xdebug

## Sample Opening Questions

- "Tell me about your PHP experience. What frameworks have you worked with?"
- "Can you explain the difference between include, require, include_once, and require_once?"
- "How do you prevent SQL injection in PHP?"
- "What's your experience with Laravel or Symfony?"

## Red Flags to Watch For

- Using outdated PHP features (PHP 5.x syntax)
- Not understanding security vulnerabilities (SQL injection, XSS, CSRF)
- Poor understanding of OOP principles
- No experience with modern frameworks
- Not using prepared statements
- Not familiar with Composer
- Poor error handling practices

## Green Flags to Recognize

- Experience with PHP 8.x features
- Strong Laravel/Symfony knowledge
- Understanding of PSR standards
- Security-conscious coding
- Testing and TDD experience
- Knowledge of design patterns
- Experience with caching (Redis, Memcached)
- Understanding of performance optimization
- Familiarity with Docker and containerization
- Knowledge of API development and authentication (OAuth, JWT)

Remember to adapt question difficulty based on the candidate's experience level and the role requirements.
